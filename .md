# SkillSwap Lite

A simple Fiverr-like service marketplace landing website that demonstrates a full-stack web application with Node.js, MongoDB, and vanilla JavaScript.

## Features

- 🌓 Dark/Light mode toggle
- 🔍 Search with autocomplete suggestions
- 👤 Email/Password authentication
- 🔐 Google OAuth integration
- 💼 Service provider profiles
- ⭐ Rating system
- 📱 Responsive design
- 🎯 Clean and beginner-friendly code

## Tech Stack

- **Frontend**: HTML, CSS, Vanilla JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose
- **Authentication**: JWT, Passport.js (Google OAuth)
- **Other**: bcryptjs, cors, dotenv

## Installation

1. Clone the repository
2. Run `npm install`
3. Create a `.env` file with your MongoDB URI and Google OAuth credentials
4. Run `npm run dev` for development
5. Open `http://localhost:3000`

## Deployment

- **Backend**: Deployed on Render
- **Frontend**: Deployed on Netlify
- **Database**: MongoDB Atlas
